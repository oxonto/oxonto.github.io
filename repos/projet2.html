Nom du Projet

Ce script en Bash premet de, une fois votre distribution Linux installée :

- Mettre à jour votre système

- Ajouter quelques bons outils

- Ajouter un motd personalisé

- Modifier le fichier bashrc

- Modifier le fichier nanorc

- Créer une clés SSH sur demande

Technologies utilisées

Bash
